超連結 (Link)

在HTML中，超連結是使用<a>標籤來建立的。<a>標籤的主要屬性是href，用於指定連結的目標URL。

基本語法


        <a href="目標URL">連結文字或內容</a>

範例

以下是一個簡單的超連結範例：


        <a href="https://www.example.com">訪問範例網站</a>

這將顯示為：訪問範例網站

其他屬性

  • target: 指定連結打開的方式。例如，target="_blank"會在新視窗或分頁中打開連結。
  • title: 提供有關連結的額外資訊，當用戶將鼠標懸停在連結上時會顯示此資訊。
  • rel: 定義當前文件與被鏈接文件之間的關係，例如rel="noopener"可以提高安全性。

範例：帶有其他屬性的超連結


        <a href="https://www.example.com" target="_blank" title="前往範例網站" rel="noopener">訪問範例網站</a>

這將顯示為：訪問範例網站
